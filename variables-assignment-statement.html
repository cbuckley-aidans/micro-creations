Lesson 02

Variables: The Assignment Statement

Data Type 01
str
Text
Used for words, sentences, and any other characters. Text values are always wrapped in quote marks so Python can tell them apart from variable names.
"hello"   "Alex"   "year 7"
Data Type 02
int
Whole number
Used for counting things: scores, ages, years. There is no decimal point. No quote marks needed because Python already knows it is a number.
100   -5   0   2024
Data Type 03
float
Decimal number
Used when you need a number with a decimal point: measurements, temperatures, prices. Like int, no quote marks are needed.
3.14   9.81   36.6
