From flowchart to if / else

Turning a decision diamond into code.

First the shape of the statement, then five of them together.

if

The shape of it

Five parts. Every if / else has them.

if temperature > 30:
print("hot")
else:
print("ok")

The word if begins the question.

Then the condition. A true or false question.

A colon ends each question line.

Indented lines belong to the if. They run only when it is true.

else is the otherwise. It runs when the condition is false.
