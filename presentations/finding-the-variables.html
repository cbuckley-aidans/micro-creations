The problem

At the school fete, you are running the lemonade stall. You want the micro:bit to count each cup you sell. When you press button A, it should show how much money you have made. Each cup costs $2.

You've seen the lemonade stall before.

This time I'm not looking for jobs. I'm looking for anything the program has to remember or work out.

2

What is a variable?

A variable is something the program needs to remember or work out.

For each one I find, I ask four questions:

  • What does it store?
  • What will I name it?
  • What is an example value?
  • Does it change?

A special kind of variable

A value that never changes is a constant.

The price of a cup is set once and stays the same. We still store it, but it never gets updated.

Spotting the difference between what changes and what stays the same is the real skill here.
