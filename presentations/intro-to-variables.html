0

New concept

Variables

How programs store and remember information.

We will start from the very beginning.

1

What is a variable?

A variable is a named box.

score

It stores one piece of information your program can use later.

This box is named score. Right now it holds 10.

The name lets you find it. The value is what is inside.

2

Making a variable

score = 0
score

This makes a box called score and puts 0 inside.

The name goes on the left. The value goes on the right.

The = sign means "put this value into the box".

3

What can a variable hold?

Numbers and words.

age = 13

A number is written on its own.

name = "Ada"

Words, or text, go inside quote marks. The quotes tell the program this is text, not a number.

4

Variables in maths

You can calculate with them.

score
score + 5

score holds 10, so this works out to 15.

A variable is used just like the value it holds.

5

Changing a variable

A variable can change. That is the whole point.

score = 10
score

Give it a new value, and the old one is replaced.

The 0 is gone. A box only ever holds one value at a time.

6

Naming your variables

Choose names that explain themselves.

Good names say what they hold: score, age, total.

score is much clearer than something like x.

Two rules: no spaces, and start with a lowercase letter.

If a name needs two words, join them with an underscore: high_score.

7

Recap

Three things to remember.

  1. A variable is a named container.
  2. The name stays; the value can change.
  3. One name holds one value at a time.
